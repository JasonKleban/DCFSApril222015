Why is functional programming relevant?

Functional programming encourages thinking about:

  • purity
  • immutability
  • composition
  • side-effects
  • code-as-data

In turn, these concepts will lead you to:

  • parallelizable,
  • distributable,
  • simpler,
  • testible,
  • maintainable
  • correct code
